기능 살펴보기 방법을 설명하기 위한 마커입니다. 다른 마커를 선택하시면, Firefox 기능을 설명해 드립니다.

탭을 상위로

탭을 주소창 위에 놓을 수 있어서 방문 중인 웹 사이트의 콘텐츠에 집중할 수 있습니다.

앱 탭

웹 메일이나 소셜 네트웍 사이트 처럼 자주 가는 사이트는 항상 고정 시킬 수가 있습니다.

새 탭

Firefox는 새로운 웹 페이지 보기와 작업을 빠르게 할 수 있습니다. + 버튼을 누르면 새 탭을 바로 열 수 있습니다.

탭 바로가기

새 탭을 열기 전에 Firefox가 이미 그 웹 사이트를 탭으로 열고 있는지 확인해서 바로 이동시킬 수 있습니다. 탭을 다시 열 필요가 없습니다.

Firefox 메뉴 버튼

여러분이 자주 쓰는 메뉴들만 모아서 쉽게 실행 할 수 있게 하나의 버튼에 넣었습니다.

북마크 버튼

오른쪽 상단에 하나의 북마크 버튼만으로 모든 것을 할 수 있습니다.

홈 버튼

홈 버튼은 브라우저 오른쪽에 있으며, 처음 시작할 때 뜨게 하고 싶은 홈페이지 목록들을 관리 할 수 있습니다.

스마트 주소창

주소창에 URL 일부만 치더라도 여러분이 방문한 웹 사이트를 바로 볼 수 있어 기억이 안나거나 북마크에 저장하지 않아도 바로 갈 수 있습니다.

웹사이트 확인 기능

웹 사이트 방문 전에 중요 정보 및 신원 확인을 하시려면 주소창의 파비콘을 클릭하시면 바로 보실 수 있습니다.

맞춤형 도구 모음

여러분은 Firefox의 인터페이스를 원하는 대로 재배치 하거나 버튼을 추가 삭제할 수 있습니다.

부가 기능 관리자

부가 기능 관리자는 이제 탭으로 열리며, 검색 및 추가 삭제를 쉽게 할 수 있도록 바뀌었습니다.

테마 기능

Firefox에서 클릭 한번 으로 테마를 바꿀 수 있습니다. 손쉬운 테마인 페르소나를 이용해 보십시오.
