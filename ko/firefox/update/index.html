Firefox 소프트웨어 업데이트가 뭔가요?

Firefox 소프트웨어 업데이트는 설치된 Firefox 브라우에 적용되는 작은 분량의 코드를 빠르게 다운로드 받습니다. 이러한 작은 패치는 사용자가 Firefox의 최신 버전을 사용할 수 있게 하는 보안 업데이트나 작은 브라우저의 변경사항을 포함하고 있습니다.

Firefox를 어떻게 업데이트 하나요?

Firefox는 업데이트를 체크하고 사용자에게 알려주도록 설정이 되어 있습니다. 알림이 나왔을 때 확인 버튼만 누르면 최신버전이 다운로드되고 컴퓨터에 설치됩니다. 새 버전을 사용하려면 Firefox를 재시작 해야 합니다.

업데이트를 직접 확인하는 것도 가능합니다. 자세한 정보는 가이드를 확인해 보세요.

현재 사용중인 Firefox 버전이 무엇인지 어떻게 확인하나요?

사용하고 있는 운영체제에 따라서 조금씩 다릅니다. 사용하는 컴퓨터별 방법을 살펴보세요.

업그레이드 하기 전에 기존의 Firefox를 지워야 하나요?

아니오. 새 버전의 Firefox는 이전에 설치된 Firefox를 자동으로 교체합니다. 업그레이드를 시작하기 위해서 오른쪽에 있는 녹색 다운로드 버튼을 누르세요.

왜 업데이트를 해야 하나요?

커뮤니티에서 개선 방향을 찾고 최신의 보안 위협을 조정하면서 Firefox는 지속적으로 진화합니다. Firefox를 최신의 상태로 유지하는 일은 가장 작으면서도 빠르고 안전한(가장 중요한) Firefox를 사용할 수 있게 하는 중요한 일입니다.

업데이트는 얼마나 오래 걸리나요?

길지 않습니다. - 전체 과정은 단지 일이분 정도만 걸립니다. 설치 절차를 도움말 페이지에서 확인해 보세요.

업데이트가 내 설정과 즐겨찾기에 영향을 미치나요?

아니오. Firefox 업데이트는 즐겨찾기나 저장된 비밀번호, 다른 설정에 영향을 미치지 않습니다. 하지만 업데이트 직후에 일부 부가기능은 호환이 되지 않을 가능성이 있습니다.

이미 Firefox를 사용하고 있었는데 이를 새로 설치하면 내 기존 설정은 어떻게 되나요?

전혀 없습니다. Firefox를 재 설치해도 즐겨찾기나 설정이 변경되지 않습니다.

Firefox 업데이트는 안전한가요?

그렇습니다 – 우리는 사용자의 안전에 높은 우선순위를 두고 있습니다. 우리는 내부에서 최신의 보안 위협을 계속해서 모니터링하는 전문가 그룹을 운영하는 오픈소스 보안 프로세스는 운영하고 있습니다. 보안 위협이 발견되지마자 이를 즉시 방지하기 위한 패치와 업데이트를 적용합니다. Firefox 업데이트를 다운로드 받아야 안전한 온라인 활동을 계속해서 할 수 있습니다.

업데이트를 하지 않으면 어떻게 되나요? 내가 직접 업데이트를 확인할 수 있나요?

네. Firefox는 자동으로 업데이트를 확인하게 되어 있습니다. 하지만 직접 업데이트를 확인하는 일도 가능합니다.
