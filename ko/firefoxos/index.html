오픈 웹 모바일 운영체제

새롭고 강력한 개방형 모바일 세상이 옵니다!

Firefox OS는 모바일 기기를 위한 오픈 웹 플랫폼입니다. 새로운 웹 표준의 개발을 통해 혁신을 주도하고 있습니다.

새로운 웹 표준

Firefox OS는 기존의 상용 모바일 플랫폼에 있는 제한 및 폐쇄성에서 자유로운 플랫폼 경쟁이 가능하도록 디바이스 독립적인 새로운 웹 표준을 만들고 구현하고 있습니다.

상용 모바일 플랫폼으로 부터 자유

우리는 기기 제조 업체와 통신사와 직접 협력하여 사용자와 시장의 요구 사항에 맞추어 제품을 제공하고자 합니다. 사용자와 개발자는 더 이상 특정 플랫폼에 갇혀 있을 필요 없으며 앞으로 다양한 디바이스와 앱을 넘어 정보를 공유하고 제공할 수 있습니다.

제조 업체와 통신사와 협력

스마트폰 제조 업체와 통신사는 운영 체제와 관계 없이 각자의 콘텐츠와 역량으로 경쟁할 수 있습니다. 앞으로 사용자의 주목을 얻기 위해 사용자 경험을 확장하고 각자 앱 경험과 배포 및 지불 수단을 통해 자유로운 경쟁이 가능해집니다.

개발자에게 기회 제공

Mozilla가 제안하는 기기용 Web API와 HTML5을 이용하면 개발자들은 네이티브앱 수준의 완전히 새로운 사용자 경험을 제공할 수 있습니다. 웹 개발자들은 더이상 상용 플랫폼의 네이티브 API를 배울 필요가 없습니다.

소비자 자유 제공

Firefox OS 기기를 사용하는 일반 고객들은 특정 플랫폼에 갇히지 않고, 선택과 유연성 및 자유를 얻게 됩니다. Firefox OS는 바로 웹을 플랫폼으로 만드는 새로운 시작입니다.
